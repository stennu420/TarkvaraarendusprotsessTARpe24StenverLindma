UML


Mis on UML?

UML ehk (Unified Modeling Language) on graafiline keel, mida kasutatakse tarkvara ja süsteemide visualiseerimiseks, kavandamiseks ning dokumenteerimiseks. See aitab arendajatel ja ärianalüütikutel keerulisi süsteeme paremini mõista.

UML diagrammide indeks

  • Flowchart / Activity Diagram
  • ERD (Entity Relationship Diagram)
  • Use Case Diagram
  • Class Diagram
  • Sequence Diagram
  • State Diagram
  • Component Diagram

Use Case Diagram

Use Case Diagram ekh (kasutusjuhtude diagramm) see on visuaalne UML-diagramm,mis kirjeldab süsteemi funktsioone. See näitab, kuidas kasutajad või süsteemid kavandatav tarkvaraga suhtleb.

Elemendid:
  • Actot(tegutseja)
  • Use Case (kasutajjuht)
  • System Boundary(süsteemi piir)
  • Association (seos)
  • Include (kaasamine)
  • Extend (laiendamine)
  • Generalization (üldistamine)
[Kasutaja] --> (Logi sisse)
[Admin] --> (Halda kasutajaid)
